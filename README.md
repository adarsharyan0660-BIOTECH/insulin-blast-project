# 🧬 Comparative Analysis of Human Insulin Gene using BLAST

## 📌 Overview
This project analyzes the human insulin gene using BLAST from NCBI to study similarity across different species.

## 🔬 Tools Used
- NCBI
- BLAST

## 📊 Results
The analysis showed high similarity with primates like chimpanzee, gorilla, and orangutan.

## 🧠 Conclusion
The insulin gene is highly conserved across species, showing its biological importance.
